Gamer Time

Gamer time started out merely as an idea. As someone who learned programming on my own and has always pushed myself to learn, it was merely a project for me to learn SQL. It turned into way more than that. I expanded on my tools I learned making the League of Legends Win Predictor, making this my biggest project yet. I continued to learn how to use Flask and Bootstrap at a higher level. I learned how to use PostgreSQL, Redis, Azure technologies for deployment, and Google Sheets API. Throug these technologies, I was able to make the program above, which has helped us at UCSD Esports keep track of who uses the TEC Cafe, when they use it, and for how long, all directly put into a Google Sheet everynight.
